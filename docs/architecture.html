Modules

Platform Detector

src/content/detector.ts

Matches hostname to platform config. Falls back to MutationObserver-watch when the chat container hasn't rendered yet.

DOM Scraper

src/content/scraper.ts

Extracts assistant text via MutationObserver + 500ms polling. innerText for Gemini, textContent for ChatGPT. Fires initial-text callbacks for existing DOM content.

Token Estimator

src/content/estimator.ts

Char-based heuristic (~4 chars/token) with per-platform multipliers (1.3× ChatGPT, 1.5× Gemini, 1.4× Claude, 1.5× Perplexity). Returns diff from lastTokenCount to prevent double-counting.

Water Converter

src/content/converter.ts

Fixed ratio: 0.003 ml per token. Based on published literature for LLM inference water consumption.

Bottle Overlay

src/content/overlay.ts

Canvas 2D pixel-art water bottle (16×28 grid). setInterval 16ms render loop. Per-cell water shimmer, cork pop animation at 95% capacity, right-click capacity setting.

Conversation Tracker

src/content/tracker.ts

Manages conversation lifecycle: start, resume (by title + platform), addDelta. Caps at 9,999L. Persists to chrome.storage.local on every delta.

Storage

src/shared/db.ts

chrome.storage.local wrapper (class named IndexedDBStore for historical reasons). CRUD operations for ConversationRecords. Shared across all extension contexts.

Dashboard

src/dashboard/

Standalone page (separate Vite build). Opens on bottle double-click. Donut chart by platform, bar chart by date, topic categorizer, human-day equivalents.

Orchestrator

src/content/index.ts

WaterCalculator class wires all modules together. Handles SPA navigation (history monkey-patching + URL polling), tab visibility lifecycle, and callback URL guards.

Service Worker

src/background/index.ts

Message relay. Platform config holder. Opens dashboard tabs on OPEN_DASHBOARD message. Entry point: background build target.
